الجمعية الدائمة للامتحانات العامة

الدراسة : الإعدادية / العلمي والأدبي

المادة : التربية الإسلامية

أحكام التلاوة ( ١٠ درجات )

س ١ : قال تعالى : ( وَلِيَكْتُبُ بِيَنْكُمْ كَاتِبٌ بِالْمَدْلِ وَلَا يَأْتِ كَاتِبٌ أَنْ يَكْتُبَ كَمَا عَلَّمَهُ اللَّهُ )

( في قلوبهم قرشٌ فَرَادَهُمْ اللَّهُ مَرْضًا وَلَهُمْ عَذَابٌ أَلِيمٌ )

عَيْنٌ ما ورد في الآيتين الكريمتين من أحكام النون الساكنة والتنوين ثم اذكر نوعها والسبب . ( اختر خمسة )

الحفظ ( ٢٠ درجة )

س ٢ : أجب عن أولاً أو ثانياً :

أولاً : اكتب من سورة النساء من قوله تعالى : ( يَكْتُبُهَا النَّاسُ أَثْقُوا رَبَّكُمْ ... إلى قوله ... فَكُلُوهُ هِيَكُمْ رَبَّهَا )

واضبط أو اخر الكلمات بالشكل .

ثانياً : من سورة ( آل عمران ) اكتب الآيات التي تجود بالمعاني الآتية :

( ١ ) الذين يكفرون بعد إيمانهم ويزدادون في كفرهم فإن الله لن يقبل من أحدهم فدية .

( ٢ ) كل الأطعمة الطبية كانت حللاً لأبناء يعقوب ولما نزلت التوراة حرم الله على بني إسرائيل بعض

الأطعمة . ومن كذب على الله بعد قراءة التوراة فأولئك هم الظالمون .

الفهم والمعاني ( ٢٠ درجة )

س ٣ : أولاً : أجب عن ثلاثة أفرع ولكل فرع ( ٥ ) درجات .

( أ ) قال تعالى : ( إِنَّ الَّذِينَ يَكْتُبُونَ مَا أَنزَلْنَا مِنَ الْبَيِّنَاتِ وَالْهُدَى مِنْ بَعْدِ مَا بَيَّنَاهُ لِلنَّاسِ فِي الْكِتَابِ أَوْلَائِكَ

يَلْعَمُونَ اللَّهُ وَيَلْعَمُونَ الْمُجْتَمِّرَاتِ ) ما البينات التي يخفونها ؟ ومن يكتمها ؟ وما جزاؤه ؟ ومن استثنى ؟

( ب ) قال تعالى : ( وَلَا تَقْرَبُوا الَّذِي إِنَّهُ كَانَ فَنِحْتَهُ وَسَاءَ سَبِيلًا )

الزنى فعلة فاحشة بنس مسلكاً ، لماذا ؟ وبم أمرنا الله ؟ وما عقوبة الزاني ؟ وما عقوبة الزاني المتزوج ؟

( ج ) قال تعالى : ( وَمَا أَرْسَلْنَا فَبَلَكَ إِلَّا رِجَالًا نُوحِيَ إِلَيْهِمْ فَاسْتَوَارًا أَهْلَ الْذِكْرِ إِن كُنتُمْ لَا تَعْلَمُونَ )

وَمَا جَعَلْنَاهُمْ جَسَدًا لَا يَأْكُلُونَ الطَّعَامَ وَمَا كَانُوا خَالِدِينَ ( ٨ )

ماذا أرسل الله قبل محمد ( صلى الله عليه وآله وصحبه ) ؟ ومن هم أهل الذكر ؟ وما صفة أولئك المرسلين ؟

وما وَعَدَ الله لهم ؟

( د ) قال تعالى : ( مِّنْ نَفْسٍ عَلَيْكَ بِأَنَّهُم بِالْحَقِّ إِنَّهُمْ فِتْنَةٌ مَاسَوُا بِرَبِّهِمْ وَزِدْنَاهُمْ هُدًى )

وضح أهم وأبرز ما يستنتج من قصة أصحاب الكهف .

( ه ) قال تعالى : ( وَلَا تَفْقُ مَا لَيْسَ لَكَ بِهِ عِلْمٌ إِنَّ السَّمَعَ وَالْبَصَرَ وَالْفُؤَادَ كُلُّ أَوْلَائِكَ كَانَ عَنْهُ مَسْتَوَلًا )

ما تفسير الآية الكريمة ؟ و إلام ترشدنا ؟

ثانياً : أعط معاني خمس مما يأتي :

الضالون ، وبث منهما ، إنه كان حوياً كبيراً ، يقدر ، القسطاس ، أسروا النجوى ، أضغاث أحلام

الحديث الشريف / ١٥ درجة

س ٤ : ( ١ ) اكتب حديثاً شريفاً في ( حُسن الخُلق ) .

( ٢ ) لم يترك الإسلام صغيرة ولا كبيرة في كل ما يؤدي بالمسلم إلى الخلق القويم إلا ودعا إليها ، وضحها .

( ٣ ) أوصى الرسول الكريم بالنساء وقد شدد الوصية بالمرأة اعترافاً بأهمية المرأة في المجتمع وفي بناء الأسرة ،

وضح ذلك .

التهذيب ( ١٥ درجة )

س ٥ : أجب عن أولاً أو ثانياً :

أولاً : ما منشأ الغضب ؟ وما قسماه ؟ وكيف يكتسب الإنسان فضيلة كظم الغيظ ؟

ثانياً : وضح ثلاث سمات من سمات الرجولة .

الأبحاث ( ٢٠ درجة )

س ٦ : أجب عن فرعين :

( أ ) للدولة الحق في تحديد الأسعار في ظروف معينة ، ما هي ؟

( ب ) اهتم القرآن الكريم بتنظيم أحكام الأسرة فجعل بنيان الأسرة على أسس ثابتة ، ما هي ؟

( ج ) من حقوق الزوج ( الطاعة فيما لا معصية فيه الله ) ، وضح ذلك .
